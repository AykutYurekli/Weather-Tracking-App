# Weather Tracking App

Bu basit Python uygulaması, şehir bazlı hava durumunu takip etmenizi sağlar.  
Girilen şehrin sıcaklığı, hissedilen sıcaklığı, nemi ve hava durumu açıklamasını görürsünüz. Ayrıca duruma uygun bir ikon da gösterilir.

## Neler Yapabilirsiniz?

- Şehir adı girip anlık hava durumunu öğrenmek  
- Sıcaklık ve hissedilen sıcaklığı görmek  
- Havanın bulutlu, güneşli, yağmurlu veya kapalı olduğunu ikonlarla görmek  

## Gerekenler

- Python 3.x  
- `requests` kütüphanesi
- Tkinter (Python ile birlikte gelir)
